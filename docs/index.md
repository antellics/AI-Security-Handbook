# GenAI Offensive Security Handbook

### Comprehensive Security Testing Standard and Pentesting Field Manual for AI-Enabled Applications

**Version:** 1.0
**Audience:** AppSec Engineers, Penetration Testers, Red Teams, Security Researchers, Security Architects

---

## About This Handbook

Generative AI systems introduce a new semantic attack surface in modern software architectures. This handbook provides a structured, repeatable methodology for testing applications that incorporate Generative AI capabilities — including Large Language Models (LLMs), Retrieval-Augmented Generation (RAG) systems, and AI agents.
